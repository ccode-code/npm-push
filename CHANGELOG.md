## [0.1.1] - 2026-01-15

### 新增

- 新增: 🎯 **支持 Monorepo 和单仓库**：自动检测项目类型，支持 workspace 模式
- 新增: 📦 **交互式发布流程**：友好的命令行交互界面
- 新增: 🔄 **自动化版本管理**：支持语义化版本（semver）
- 新增: 🏷️ **Git Tag 管理**：自动创建和推送 git tag
- 新增: 📝 **Changelog 支持**：发布前记录变更日志，自动生成 CHANGELOG.md 文件
- 新增: 🛠️ **脚本执行**：支持发布前执行构建脚本（如 build）
- 新增: 🎨 **美观的 UI**：带有 loading 动画和成功提示
- 新增: 📚 **JS API**：同时提供程序化 API 供 Node.js 脚本使用
- 新增: 🌍 **多语言支持**：支持中文、英文、日语、韩语，自动检测系统语言
